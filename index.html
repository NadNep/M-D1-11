D1.11

Инструкция по заданию

Подготовьте данные для авторизации в Trello, вам понадобятся key, token и КОРОТКИЙ ID вашей доски

Получить короткий ID можно из адреса странички необходимой доски. https://trello.com/b/КОРОТКИЙ_ID


Варианты запуска программы:

  • без параметров - программа выводит на экран колонки и задачи в них
  • с параметром "create имя колонка" - создаст задачу _имя_ в _колонке_
  • с параметром "move name имя колонка" - переместит задачу _имя_ в _колонка_, если задач несколько, предложит выбор
  • с параметром "listadd колонка" - создаст колонку _колонка_
